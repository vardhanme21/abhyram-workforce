# Abhyram Workforce Portal

A premium workforce management system built with Next.js and Salesforce.

## Project Structure

- **workforce/**: Next.js frontend application (App Router, TypeScript, Tailwind CSS).
- **abhyram-backend/**: Salesforce DX project (Apex Classes, Custom Objects, Permission Sets).
- **DEPLOYMENT_GUIDE.md**: Step-by-step instructions for deploying to Vercel and Salesforce.

## Quick Start

1.  **Backend**: Deploy `abhyram-backend` to your Salesforce Org.
2.  **Frontend**: Navigate to `workforce`, install dependencies (`npm install`), and run `npm run dev`.

See `DEPLOYMENT_GUIDE.md` for full setup details.
